3U VPX Development Chassis

BCube's 3U VPX Development Chassis 3010 is a 3U Single Slot VPX Backplane for Vita 46 Payload Module with support for external connectors includes ethernet and Others feature as described below.

Typical applications include

  • Conduction cooled
  • Supports VITA 46 connectivity
  • Supports VPX standard Power Input Ports
  • Supports JTAG Line + SM Bus Management + On board Geographical Address
  • Supports x4 PCIe Data Plane Port
  • Supports 1G Base T Ethernet Control Plane Ports
  • Supports Expansion Ports IOs on Standard connectors
  • Supports User Defined IOs on Standard Connectors
